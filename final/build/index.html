TRIGGER:

  • - Scroll down

RULES:

  • On Scroll

  • - Image fades in from 0% opacity to 100%, in 1 second

  • - No fading out and fading back in if image is already faded in

  • On Hover

  • - User can hover over an image only if the image is faded in 100%

  • - Cursor changes to finger pointer when hovering over image

  • - Modal with clothing item name and price slides up from the bottom of the image when hovering over image

  • - Modal with clothing item name and price slides down towards the bottom of the image and disappears when no longer hovering over image

  • - 'Quick View' button changes to black background color and white text on hover

  • On Click

  • - Modal with clothing item information, color options, size options, and 'add to cart' button appears; expands from middle of the page

  • - Everything behind the modal is darkened by a dark gray background with 60% opacity

  • - Cursor changes to finger pointer when hovering over buttons within the modal

  • - Individual color option circle buttons have thicker borders when user hovers over them

  • - Individual size option rectangle buttons are filled in with light gray when users hover over them

  • - Individual clothing item images on the left side of the modal have black borders when user hovers over them; when image is clicked, the border will stay around the image

  • - User can click the 'X' button on the upper right corner of the modal to exit the modal
